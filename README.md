# React Steps Form

Simple dish ordering steps form using react

## Used
* React 16
* Ant Design
* Axios
* Express Server
* Joi Validator

## Installation

* `git clone git@github.com:symaster1995/react-steps-form.git`
* cd zenport-order
* npm install
* npm run dev
* visit `http://localhost:3000/`
